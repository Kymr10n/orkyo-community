# Roadmap

Orkyo Community is in active development and usable today, but still pre-1.0 — expect occasional breaking
changes between minor versions (documented in [CHANGELOG.md](CHANGELOG.md)). This roadmap is direction, not
a dated commitment; priorities shift with real-world feedback.

## Near-term

- **Availability & absence model** — richer working-hours, absences, and off-time handling.
- **Notifications & reminders** — email/notification hooks for upcoming and changed assignments.
- **Calendar / iCal integration** — subscribe to schedules from external calendars.

## Under consideration

- Deeper reporting and export from Insights.
- Import helpers for onboarding existing schedules.
- Additional resource types beyond spaces and people.

## How priorities are set

Work is weighed against real usage and the [foundation placement rule](.github/CONTRIBUTING.md) (shared
behaviour lands in [orkyo-foundation](https://github.com/Kymr10n/orkyo-foundation); single-tenant/self-host
specifics stay here). To influence the roadmap, [open an issue](https://github.com/Kymr10n/orkyo-community/issues)
or start a [discussion](https://github.com/Kymr10n/orkyo-community/discussions) — concrete use cases help most.
